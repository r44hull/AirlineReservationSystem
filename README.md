# Airline Reservation System

This project is a **Java-based Airline Reservation System** developed as a college mini project. The system demonstrates basic airline booking operations such as managing flight information, passenger details, and reservation records using Java.

## Project Structure

* **src/** : Contains all Java source code files for the application.
* **nbproject/** : NetBeans project configuration files.
* **build/classes/** : Contains compiled `.class` files generated after building the project.
* **build.xml** : Ant build configuration file used to compile the project.
* **manifest.mf** : Manifest file containing metadata for the Java application.

## Features

* Manage airline reservations
* Store passenger information
* Handle flight booking records
* Demonstrates object-oriented programming concepts
* Simple project structure suitable for learning Java development

## Technologies Used

* Java
* NetBeans IDE
* Apache Ant (build tool)

## How to Run the Project

1. Clone or download the repository.
2. Open the project in **NetBeans IDE**.
3. Build the project using the build option.
4. Run the project from the IDE.

Alternatively:

1. Navigate to the project directory.
2. Compile the project using the build tool.
3. Run the compiled Java application.

## Notes

This project was developed for educational purposes to understand **Java programming, object-oriented design, and basic project structure**.
